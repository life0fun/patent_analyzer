# Claim Parser Skill

**ID**: `extract_features`
**Description**: Decomposes a patent claim text into distinct, analyzable features (elements/steps).

## Inputs
- `claim_text` (string): The full text of the patent claim.

## Output
- JSON string (List[str]): A list of string features extracted from the claim.

## Tool Mapping
Maps to MCP Tool: `extract_features`
